************
Installation
************

Install with `pip`:
-------------------

.. code-block:: sh

   pip3 install uptick

Manual installation:
--------------------

.. code-block:: sh

   git clone https://github.com/xeroc/uptick
   cd uptick 
   python3 setup.py install --user

Upgrade
-------

.. code-block:: sh

   pip3 install uptick --user --upgrade
